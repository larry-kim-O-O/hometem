
# ホームテム – 無料ホスティング向け静的サイト

本リポジトリは、Amazonアソシエイトの主要ポリシーに配慮した静的サイト雛形です。価格表示はPA-APIまたは公式ウィジェット利用時のみ行う設計で、初期状態では価格を表示しません。

## 無料ホスティング手順（例）
### GitHub Pages
1. GitHubで新規リポジトリ作成（Public）
2. 本ファイル一式をアップロード
3. Settings → Pages → Branchを`main`/`root`に設定→保存
4. 公開URLを`robots.txt`と`/sitemap.xml`のドメインに反映（search/replace）

### Vercel / Netlify
- いずれも無料プランでデプロイ可能。リポジトリ連携→Deploy。

## ドメイン置換
- プロジェクト内の`https://homutemu.example`をあなたの実ドメインに置換してください。

## Amazonアソシエイトの注意点
- 価格・在庫は**PA-APIまたは公式ウィジェット**のみ。表示には**取得時刻の明記**が望ましい。
- **価格の手動入力・スクリーンショット掲載は不可**。
- 特別リンクのクリックに対する**インセンティブ提供は禁止**。
- メール/DMでのリンク共有は**オプトイン済み**の受信者に限定。
- 短縮URL等でトラッキングが不明瞭になる導線は避ける。

## PA-API連携（任意）
- サーバレス関数（Vercel/Netlify Functions）で署名付きリクエストを実装し、`/posts/*`に価格ブロックを埋め込むと運用が容易です。
- 連携前でもリンクボタンのみで運用可能です（価格非表示）。
